A WordPress plugin That connect your client directly with your agent
== Description ==

A WordPress plugin that can connect third party app with your website just install this plugin and see all the results you just need to do some minor changes as mention in callback24 setting page

== Features ==

1. Connect your sales agent with your client directly

== Installation and Use ==

This section describes how to install the plugin and get it working.

- Upload the plugin files to the directory, or install the plugin through the WordPress plugins screen directly.
- Activate the plugin through the ‘Plugins’ screen in WordPress
- Your block patterns will be available to select from Dashboard > Click on the plus icon in the editor

== Frequently Asked Questions == 

- Step 1. Check if your Plugin is activated.
- Step 2. Deactivate all plugins and reactivate Block Patterns Collection.
